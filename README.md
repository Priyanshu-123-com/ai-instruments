# AI Instruments

A collection of virtual musical instruments powered by AI.

## Files

- `virtual_guitar.py` - Virtual guitar implementation
- `virtual_piano.py` - Virtual piano implementation
- `virtual_drum_set.py` - Virtual drum set implementation
- `requirements.txt` - Python dependencies

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the desired instrument script

## Usage

Run any of the Python files to start the virtual instrument.
